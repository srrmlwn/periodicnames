I want to create a web app called periodicnames.com.

The idea is to let the user find their names (or really any other word) using the element symbols from the periodic table of elements.

The code base already has some code i wrote for the projects more than a decade back, that can act as a starting point.

Here's everything I want to do with this side project.

1. Let users find their names (or anything else) in the periodic table of elements. Use a sleek CX and show some snappy animation to let the users know which elements we are selecting .
2. Ability to let the user share the results on social media - e.g., instagram, twitter, tiktok etc. Initially it can be just a post with an image of the result, but eventually maybe reels with an animation of the user finding their name in the periodic table.
3. Link to some external service (say Shopify print on demand, or printify) to let the user print the results on a t-shirt, mug, mouse pad etc etc. This is how i seek to make some sales using periodicnames.com.
